GPU Scroll Experience

Product motion controlled by scroll.

This project documents my first hands-on build with scroll-controlled image sequences: generating motion frames with Google Flow, slicing them into assets, and turning them into an interactive GSAP canvas experience.

About me See the process

Learning Build

From generated frames to a scrollable web moment.

I used this GPU motion study to learn how frame sequences, pinned scrolling, canvas drawing, and responsive animation work together on a real page.

01 / Generate

Create the motion source in Google Flow.

I started with AI-generated GPU visuals, used first-and-last frame motion generation, then exported the animation into a frame sequence I could control in the browser.

02 / Build

Map scroll progress to image frames.

The page preloads every frame, pins the visual stage with ScrollTrigger, and uses scroll progress to choose the exact canvas frame.

03 / Refine

Make it feel like a polished interaction.

I added parallax depth, sticky explanation panels, hover states, mobile layout rules, and a full-screen sequence stage so the project feels like a finished product page.
